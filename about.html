Bread

On the technical side, what keeps pages and writings working together is Jekyll, a Ruby‐based static site generator, which makes it as easy as pie to actually write articles in Markdown. The entire source for this site sits on GitHub, and is open for inspection. If you find broken links or anything else you believe is a genuine error, please let me know.

Code‐wise, the Website is in valid (as much as it can be), strict, and semantic HTML5, styled with not‐so‐valid CSS3. I’ve tried to include the best Schema and ARIA authoring practices.

This site is happily crafted with Sublime Text 2 in my lovely MacBook.

Butter

I’m no designer myself, so the only thing I could really try to tackle was simplicity. I had a hard time trying to design something that I liked. I wanted a minimalist yet elegant design that emphasized the text and avoided the usual clutter of sidebars and other paraphernalia found on most blogs…

Okay, that’s what everyone would like to hear, but the truth is that I couldn’t get a sidebar working as I wanted, so I glued it all together and let it ﬂow. I’m also still trying to find a cool cross‐browser font. For some reason, my current font only displays nicely with -webkit-font-smoothing: antialiased, which luckily Google Chrome uses by default. The site works just fine without it, but it certainly loses some appeal.
